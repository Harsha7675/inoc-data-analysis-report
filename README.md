# inoc-data-analysis-report
This project focuses on analyzing INOC data to generate meaningful business insights using Power BI. The dashboard helps in monitoring key performance indicators (KPIs), identifying trends, and supporting data-driven decision-making.

🎯 Objectives :
The main goal of this project is to analyze INOC operational data and extract useful insights. The dashboard focuses on tracking key performance metrics, understanding trends over time, and helping stakeholders make better decisions using data.

🛠️ Tools & Technologies :
Power BI for dashboard development
Excel / CSV files as the data source
Power Query for data cleaning and transformation
DAX for creating calculated measures and KPIs

📂 Dataset Description :
The dataset contains operational data related to incidents and performance. It includes details such as incident records, time-based information, and various performance indicators used to evaluate overall operations.

🔄 Data Processing :
The data was initially in raw format and required preprocessing. Missing values and inconsistencies were handled, duplicates were removed, and necessary transformations were applied using Power Query. Relationships between tables were also created to support accurate analysis.

📈 Dashboard Features :
The dashboard provides a clear view of important KPIs like total incidents and resolution time. It includes interactive visuals for trend analysis along with filters and slicers, allowing users to explore the data dynamically. The layout is designed to be simple and easy to understand.

📊 Key Insights :
The analysis helped identify peak periods with higher incident volumes and provided better visibility into performance metrics. It also highlighted areas where operational improvements can be made.
